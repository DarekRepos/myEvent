myEvent Wordpress Plugin allow you to add events from WordPress widgets.

== Description ==
myEvent Wordpress Plugin allow you to add events from WordPress widgets.
All you need to do is install and configure it.

== Installation ==
1. You can either install this plugin from the Wordpress Plugin Directory, or manually [download the plugin](https://github.com/Darekrepos/myevent/releases) and upload it through the "Plugins > Add New" menu in WordPress
1. Activate the plugin through the "Plugins" menu in Wordpress

== How to Use ==
1. Go to "Dashboard > myEvent > Settings" to configure it.
1. From the menu on the left in WordPress, select the MyEvent icon and then the "Add new" option. Then you need to fill the title and metaboxes with the desired data. When finished, click the **publish** button.
1. Go to "Dashboard > Appearance > Widgets" page. Drag and drop myEvent widget to the selected sidebar.
